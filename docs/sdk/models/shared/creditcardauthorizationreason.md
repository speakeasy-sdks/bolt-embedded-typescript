# CreditCardAuthorizationReason

The reason code explaining the authorization status.
  * `1` - none
  * `2` - invalid_amount
  * `3` - invalid_cvv
  * `4` - invalid_cc_number
  * `5` - expired
  * `6` - risk
  * `7` - lost_stolen
  * `8` - call_issuer
  * `9` - invalid_merchant_for_card
  * `10` - unsupported_payment_method
