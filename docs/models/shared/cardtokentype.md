# CardTokenType

Used to define which payment processor generated the token for this credit card.

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `Vantiv`                | vantiv                  |
| `Applepay`              | applepay                |
| `Bolt`                  | bolt                    |
| `Stripe`                | stripe                  |
| `Plcc`                  | plcc                    |
| `ApplepayEncryptedBlob` | applepay_encrypted_blob |
